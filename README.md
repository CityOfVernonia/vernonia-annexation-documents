# vernonia-annexation-documents

A tool for downloading Vernonia annexation documents and converting to PDF.

Updated 2021.12.02.

### requirements

[libtiff](http://www.libtiff.org/)

### install

```shell
npm install
```

### run

```shell
node index.js
```

### use

Copy PDF files to location of your choosing.
